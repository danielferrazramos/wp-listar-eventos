== Description ==

O plugin cria um tipo de post personalizado denominado "evento" e campo personalizado para registro de data e hora do evento.

O plugin deve ser utilizado através do shortcode "[listar_eventos]". O shortcode possui um parâmetro para definir quantos eventos serão exibidos.
Utilize o parâmetro "items" para selecionar a quantidade de eventos, por exemplo, "[listar_eventos items=10].
Se não utilizar o parâmetro "items", todos os eventos serão exibidos.


= Características: =

- Campo de data e hora do evento, preenchimento obrigatório.

- Seleção de data e hora do evento utilizando datepicker nativo do Wordpress (admin).


== Installation ==

= Instalação do plugin: =

- Faça o upload da pasta wc-listar-eventos para a pasta de plugins.
- Ative o plugin.


= Configuração do plugin: =

Não existem configurações adicionais para o plugin. O shortcode "[listar_eventos]" pode ser utilizado em conteúdo de páginas e posts do Wordpress.
